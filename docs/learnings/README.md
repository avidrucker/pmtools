| [TIL 2026-07-13 CLAUDE](./today-i-learned-2026-07-13-claude.md) | 2026-07-13 | CLAUDE | Documentation does not change agent behavior — a banned-word rule recurred 5+ times across 6 agents in 8 days, twice with the agent citing the note while breaking it; only checks that run without the agent's cooperation caught anything (the linter, the NOT-ASSESSED rule, an adversarial verifier, `git cherry`, a mutation test). Re-running a command tests arithmetic, not a claim — only an independent reimplementation tests the claim. The ranking-word tell held 19/19: every failed assertion contained *highest/most/every/nobody/only/because*, and none contained a bad number. `count(*) != max(id)` is a hypothesis, not arithmetic — `AUTOINCREMENT` burns ids on failed inserts. Ask the human whether he wanted the data before writing 400 words about its loss. Report NOT ASSESSED, never a proxy — a computable-but-confounded metric is worse than a missing one, because a missing one announces itself. |
